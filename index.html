Learning Path

FAANG Interview Prep · Structured from foundations to Gen AI

01 Foundations
Data Types & Operations Python

Mutability, hash contract, is vs ==, int/float/str/bytes internals — questions asked at every FAANG screen.
Loops & Control Flow Python

Iterator protocol, generator expressions, itertools patterns — the backbone of every algorithm implementation.
Input & Output Python

Fast stdin reading, sys.stdin, formatted output — essential for competitive-style FAANG coding platforms.
02 Data Structures
Built-in Collections DS

Amortized ops, LRU Cache, Counter patterns — the tools you reach for in every interview before reinventing the wheel.
Linked Lists DS

Floyd's cycle detection, slow/fast pointer, merge K sorted lists — classic problems that appear in Google and Meta loops.
Stacks & Queues DS

Monotonic stack, min-stack O(1), sliding window max — patterns behind ~30% of FAANG array/string problems.
Priority Queues & Heaps DS

K-way merge, median of data stream, Dijkstra's shortest path — heap is the correct data structure for ~20% of hard problems.
Trees & BSTs DS

LCA, serialize/deserialize, diameter, BST validation — trees appear in every FAANG loop. Know iterative traversals cold.
Graphs & Networks DS

Kahn's topological sort, Union-Find, Bellman-Ford — graph algorithms form the hardest tier of FAANG system design interviews.
03 Algorithms
Time & Space Complexity Theory

Big-O analysis, amortized complexity, space trade-offs — every FAANG interviewer will ask you to justify your complexity claim.
Algorithms & Patterns Core

Two pointers, binary search, DP, backtracking, divide & conquer — the canonical patterns that unlock 80% of LeetCode hard.
04 Advanced Python
OOP & Design Patterns Python

SOLID, decorators, metaclasses, singleton/observer/factory — system design rounds at Amazon and Google require this cold.
File Handling Python

Context managers, mmap, pathlib — production-quality file I/O patterns tested in backend engineering interviews.
API Handling Python

Session pooling, async HTTP, circuit breaker, JWT — the patterns that distinguish a senior engineer in a system design interview.
Concurrency & Async Python

GIL internals, asyncio, thread pools, multiprocessing — critical for infra and ML platform roles at every top tech company.
05 Generative AI
What is an LLM? Gen AI

Transformer architecture, scaling laws, GQA, quantization, DPO — now a required topic at Google DeepMind, Meta AI, and OpenAI.
KV Cache Gen AI

Memory formula, FlashAttention, PagedAttention — inference optimization questions asked in ML infrastructure interviews.
RAG Gen AI

Retrieval-Augmented Generation, vector search, chunking strategies — the standard architecture for production LLM applications.
Fine-Tuning & PEFT Gen AI

LoRA, QLoRA, adapter layers, full fine-tuning trade-offs — the depth that separates ML engineers from ML scientists at FAANG.
