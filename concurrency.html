async_loop.py 
async def task():
    await asyncio.sleep(1) # yields
async def main():
    await asyncio.gather(
        task(), task()
    )

# Bypasses the GIL entirely

Synchronous

Single thread, linear run. Straightforward but blocking.

Threading

Shared memory. Perfect for I/O bounds. GIL blocks parallelism.

Asyncio

Event loop cooperative yielding. Scalable networking.

Multiprocess

Spawns independent GILs. Leverages multiple cores in parallel.
